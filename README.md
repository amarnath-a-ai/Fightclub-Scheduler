## 🔧 Customization Guide

### Adding a New Event
In `index.html`, find the `EVENTS` array and add an object:

```javascript
{
  id: 24,                          // unique ID
  org: 'ufc',                      // 'ufc' | 'one' | 'boxing' | 'bellator' | 'pfl'
  name: 'UFC 328',                 // Event name shown on card
  sub: 'Fighter A vs. Fighter B',  // Subtitle
  date: new Date('2026-08-01T23:00:00'),
  venue: 'T-Mobile Arena',
  loc: 'Las Vegas, NV',
  isTitle: true,
  titleLbl: 'Heavyweight Championship',
  broadcast: 'Paramount+',
  mainEvent: {
    f1: 'Fighter A',
    f2: 'Fighter B',
    wc: 'Heavyweight Title'
  },
  card: [
    { t: 'main',    f1: 'Fighter A', f2: 'Fighter B', w: 'Heavyweight Title' },
    { t: 'co-main', f1: 'Fighter C', f2: 'Fighter D', w: 'Welterweight' },
  ],
  result: null,     // Set to { winner: 'Fighter A', method: 'KO/TKO' } after the event
  tickets: 'Available'
}
```

### Adding a Result After an Event
Find the event in `EVENTS` and update `result`:
```javascript
result: { winner: 'Fighter A', method: 'Decision (Unanimous)' }
```

### Adding a Ticker Item
Find `UFC326_RESULTS` array and add:
```javascript
{ w: 'Winner Name', l: 'Loser Name', method: 'KO/TKO', rd: 1, time: '2:34', wc: 'Lightweight' }
```
